## Task title
squares_and_cubes

## Source
https://codebattle.hexlet.io/games/64569

## Task
Create a function that takes two numbers and checks
if the square root of the first number is equal to the cube root of the second number.


## Examples
```
true   == solution(4, 8)
false  == solution(16, 48)
true   == solution(9, 27)
false  == solution(4, 27)
```
